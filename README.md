# Siva Job Dashboard

A personal job search dashboard to track job applications, manage interview progress, and organize job search activity.

## Features

- Firebase Authentication login
- Add, edit, delete job applications
- Track application status
- Search and filter jobs
- Sort jobs by newest, company, status, applied date, and salary
- Track job source, location, job type, salary range, and applied date
- Track resume match score
- Track recruiter details, interview date, follow-up date, and next action
- Dashboard metrics for job status, response rate, average match score, interviews, and follow-ups
- Firebase Firestore database
- Firebase Hosting deployment

## Tech Stack

- React
- Vite
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
- GitHub
